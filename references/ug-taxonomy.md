# UG Taxonomy

Use this reference when deciding whether a captured page is relevant to selected-country UG competitor research.

## Relevant UG Signals

- Acquisition: invite friends, referral bonus, new-user reward, share-to-earn, first registration incentive.
- Activation: first order, first wallet bind, first recharge, first content post, first task completion.
- Retention: daily sign-in, streak rewards, task center, coins, points, missions, level progression.
- Monetization growth: cashback, coupon ladder, first purchase incentive, repeat purchase reward.
- Lifecycle recall: comeback coupons, dormant-user rewards, push/message-center incentive entry.
- Viral loop: referral code, invitation link, reward split, team/group reward, social share prompt.

## High-Confidence Page Types

- Invite/refer friends page.
- Rewards, coins, points, missions, or task center.
- Cashback or coupon center when tied to growth behavior.
- Sign-in, streak, or calendar reward page.
- Wallet/benefits page when it contains acquisition or retention incentives.
- Campaign page with explicit user action and reward mechanics.

## Usually Not Relevant

- Generic homepage banner without user-growth mechanics.
- Pure product listing, feed content, entertainment content, checkout, payment, or support pages.
- Brand campaign pages with no task, reward, invite, coupon, or retention mechanic.
- Tiny visual, timestamp, or carousel changes that do not alter conversion logic.

## Judgment Rule

Classify as UG-relevant only when the page clearly connects a user behavior to an acquisition, activation, retention, referral, or monetization-growth outcome.
